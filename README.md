# Academic Services Platform

A complete web application for providing academic services with multi-language support and admin order management system.

## Features

### 🌍 Multi-Language Support
- English
- Русский (Russian)
- Ўзбек (Uzbek)

### 📚 Services Offered
1. **Translation Services** - Professional translation in multiple languages
2. **Editing & Proofreading** - Expert editing for academic papers
3. **Administration Services** - Complete administrative support
4. **Presentation Design** - Professional presentation creation
5. **AutoCAD Design** - Technical and architectural designs
6. **Business Card Design** - Creative business card designs

### 👨‍💼 Admin Features
- Secure admin login panel
- View all customer orders
- Approve or reject orders
- Track order status (Pending, Completed, Rejected)
- Real-time order management

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Storage**: LocalStorage (Client-side)
- **Language**: Multi-language i18n system
